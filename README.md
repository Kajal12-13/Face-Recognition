# Face-Recognition
This project is a Python-based face recognition system that uses machine learning and computer vision techniques to detect and recognize human faces in real-time or from images/videos.
Features
Detects faces from images, videos, or webcam input
Recognizes known faces using facial embeddings
Real-time face matching and tagging
Built with OpenCV, face_recognition, and dlib
Easily add or remove known faces
Simple CLI or GUI integration (if applicable)

Tech Stack
Python 3.x
OpenCV
face_recognition (built on dlib)
NumPy
